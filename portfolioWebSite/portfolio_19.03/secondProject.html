Project 2 : 의류 정보 확인 시스템

대학교 2학년, C++을 이용하여 짧은 기간으로 제작했던 "의류 정보 확인 시스템" 입니다.

객체 지향 프로그래밍 언어인 C++의 기초적인 기능을 사용해서 한 개의
부모 클래스와 두 개의 자식 클래스를 이용해 프로젝트를 진행하였습니다.

프로젝트 진행 과정

프로젝트 진행은 첫째주(16.11.23 ~ 28)와 둘째주(16.11.30 ~ 12.05)로 나누었습니다.

프로젝트에 관한 자세한 내용은 "Learn More을 통해 볼 수 있습니다."

의류 정보 확인
시스템

첫째주(16.11.23 ~ 28)

첫째주에는 일반 의류 목록와 VIP고객의 개인 의류 목록을 나누어 의류 정보를 확인 할 수 있도록 나눔.

이때, VIP의류의 목록은 해당 VIP고객만 허가를 받아 확인 가능.

의류 목록의 관리는 관리자만 허가를 받아 의류를 추가 및 삭제 할 수 있도록 시스템을 설계하기로 계획을 세움.

또한, 의류 정보 확인 시에는 먼저 성별에 의한 분류 인지 브랜드에 의한 분류인지 확인하고, 다음으로 의류의 종류를 선택하도록 설계.

이 때, 선택은 해당 리스트에 있는 목록만 선택 가능 하도록 설계. VIP고객 의류정보 또한 유사한 방식.

둘째주(16.11.30 ~ 12.05)

둘째주에는 관리자 모드를 추가하여 의류의 추가 및 삭제 시스템과 비밀번호 설정 시스템을 설계.

의류의 추가 내용은 텍스트파일로 따로 데이터를 저장하여 프로그램이 종료 되어도 메모리에 저장이 되어 있도록 설계.
